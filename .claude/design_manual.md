## アセット調達ポリシー（必読）

### 基本方針
**アセットはネット上のフリー素材からダウンロードして使用する。** 自作は不要。

### ライセンス要件（必ず確認すること）
**金銭が発生する有料アセットの使用は一切禁止（社長命令）。**
判断に迷う場合は必ずURLを添えて社長に確認を取ること。

| ライセンス | 使用可否 |
|----------|--------|
| CC0 / Public Domain | ✅ 最優先（帰属表示不要） |
| CC BY | ✅ 使用可（クレジット表記が必要） |
| Free for commercial use | ✅ 使用可 |
| CC BY-NC | ❌ 商用不可のため使用禁止 |
| 「個人利用のみ」 | ❌ 使用禁止 |
| 有料・サブスクリプション | ❌ 絶対禁止 |

### 優先調達先（順番に試すこと）
| サイト | 特徴 | 主な用途 |
|-------|-----|--------|
| **kenney.nl** | 全アセットCC0。ゲーム用素材の宝庫 | スプライト・SE |
| **opengameart.org** | ゲーム特化フリー素材 | スプライト・BGM・SE |
| **pixabay.com** | アカウント不要でDL可。商用利用可 | BGM・SE |
| **soundimage.org** | ゲーム向けBGM多数。クレジット要 | BGM |
| **freesound.org** | SE特化。CC0多数 | SE |

### ファイル配置ルール
```
public/assets/
├── images/        # 画像アセット（スプライト等）
└── sounds/
    ├── bgm/       # BGM（ループ曲）
    └── se/        # SE（効果音）
```

### ファイル命名規則
- BGM：`bgm_day.mp3`、`bgm_night.mp3`、`bgm_title.mp3`
- SE：`se_wood_chop.wav`、`se_stone_hit.wav`、`se_eat.wav` 等
- スプライト：`enemy.png`、`berry.png` 等

### クレジット管理
CC0以外の素材を使用した場合は、プロジェクトルートの `CREDITS.md` に記録すること：
```
## サウンドアセット
- [ファイル名] by [作者名] / [サイト名] / [ライセンス] / [URL]
```

## サウンド実装指針

### BGM
- 昼間：明るく穏やかなループ曲
- 夜間：緊張感のある暗いループ曲
- タイトル：印象的なテーマ曲

### SE（効果音）
| SE | 発生タイミング |
|---|-------------|
| 採集音（木） | 木を採集したとき |
| 採集音（石） | 石を採集したとき |
| 食事音 | ベリーを食べたとき |
| クラフト成功音 | 焚き火をクラフトしたとき |
| 警告音 | 空腹度 30 以下になったとき |
| ダメージ音 | 敵に接触したとき（Phase 1以降） |

### 推奨フリー素材サイト
- `freesound.org`（効果音）
- `pixabay.com/music`（BGM）
- `soundimage.org`（ゲーム向けBGM）
