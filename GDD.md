# サバイバルアイランド — Game Design Document (GDD)
**バージョン：** 3.0（2026-06-05 コンセプト全面刷新）
**変更概要：** 白亜紀テーマ廃止 → 現代無人島サバイバル（リアリティ最優先設計）

---

## 1. エレベーターピッチ

> **「嵐で難破し、気づいたら無人島にいた。廃村の謎を解きながら、生き延びて脱出を目指せ。」**

現代を舞台にした2Dリアルサバイバル。  
体力・水分・体温・カロリー・睡眠・精神衛生の6軸を管理し、出血・骨折・感染症・食中毒・低体温症などのリアルな状態異常と戦いながら生き残る。  
廃村を探索して手がかりを集め、5通りの脱出ルートからひとつを選んで島を出る。

## 3. 差別化ポイント

| タイトル | リアル状態異常 | 食物連鎖 | 潮汐 | 廃村探索 | 複数END |
|---------|-------------|--------|------|--------|--------|
| Don't Starve | △ | なし | なし | なし | なし |
| The Long Dark | ○ | なし | なし | △ | なし |
| Green Hell | ◎ | なし | なし | なし | なし |
| **サバイバルアイランド** | **◎** | **◎** | **◎** | **◎** | **◎** |

**唯一無二の強み：**
1. **6軸ステータス＋リアル状態異常** — The Long Dark レベルの生存圧力
2. **食物連鎖が機能する生態系** — ウサギ乱獲 → 狼が人を狙うダイナミズム
3. **廃村の謎** — なぜ島に人が住んでいたのか？日記・手紙・電報で徐々に明かされる
4. **5通りの脱出ルート** — 難易度・プレイスタイルで自然に分岐する
5. **潮汐システム** — 干潮時のみ渡れる岩礁など、時間管理に新次元

## 4. コアループ

### 5分ループ（瞬間の行動）
```
フィールドを歩く（風向き確認）
 → 採集 / 狩猟（動物の感覚を利用）
 → 状態異常をチェック（出血中？体温低下？）
 → 緊急対応 or 次の行動
 → 水・食料の補充
```

### 30分ループ（1セッション）
```
拠点に戻る
 → 焚き火に薪を足す（濡れた薪は使えない）
 → 食料を調理（生食は食中毒リスク）
 → 廃村を探索 → 缶詰・部品・日記を発見
 → 夜になる前にシェルターへ（狼が出現）
 → 精神衛生が下がっていれば焚き火に当たる
```

### 長期ループ（数時間〜数十時間）
```
並行して複数の脱出ルートを探る：

【早期脱出】いかだルート
 素材を集め、海岸でいかだを建造 → Day 5前後に脱出可能

【中期脱出】SOS無線ルート
 廃村の発電機を修理 → 電子部品を集める → 無線機完成

【上級脱出】船・ヘリルート
 廃村の奥深くまで探索が必要 → 大量の部品 → 数十日かかる

【究極】定住ルート
 農場・水源・永続シェルターを完成させ 30日間生存
```

## 5. 世界設定

### 舞台：孤島「オリハル島」
- **経緯**：プレイヤーは嵐で難破し、見知らぬ無人島に漂着する
- **島の構造**：熱帯〜亜熱帯の島。中央に山。北東に廃村「オリハル集落」
- **廃村の謎**：かつて漁村として栄えたが、ある出来事で島民全員が消えた。その理由は日記・電報・手紙で断片的に明かされる（解明は任意）
- **時代**：現代（スマホは海に沈んだ、衛星通信も届かない孤島）

### マップ設計
```
[北] 山岳エリア（ヘリ墜落地点）
      │
[北東] 廃村「オリハル集落」（発電機・無線機・朽ちた船・医薬品）
      │
[中央] 草原・森（狩猟・採集メイン）
      │
[南西] 海岸・干潟（いかだ建造・潮汐エリア）
      │
[南東] 珊瑚礁（干潮時のみ渡れる）
```

## 6. ステータスシステム（6軸）

| ステータス | 初期値 | 最大値 | 0になると |
|-----------|-------|-------|---------|
| **体力 (HP)** | 100 | 100 | 死亡 |
| **水分** | 100 | 100 | 毎秒HP-2、脱水症で視野ぼやけ |
| **カロリー** | 100 | 100 | 毎秒HP-1、筋力低下 |
| **体温** | 37.0℃ | — | 35℃↓→低体温症 / 39℃↑→熱中症 |
| **睡眠** | 100 | 100 | 視野ぼやけ・操作精度低下・幻覚 |
| **精神衛生** | 100 | 100 | 幻聴・パニック・行動異常 |

### 各ステータスの変化要因

**水分**
| イベント | 変化 |
|---------|------|
| 毎30秒（自動） | -1 |
| 運動（走る） | -2/30秒 |
| 高体温時 | -3/30秒 |
| 海水を飲む | -20（逆効果） |
| 煮沸した川水 | +30 |
| 雨水（容器で収集） | +20 |
| ヤシの実 | +25 |
| 缶詰（汁） | +10 |

**カロリー（基礎代謝 + 活動量）**
| 食料 | カロリー | タンパク | 炭水化物 | 脂質 | リスク |
|-----|---------|--------|--------|-----|-------|
| 生ベリー | +15kcal | 0 | ○ | 0 | 食中毒10% |
| 焼きベリー | +20kcal | 0 | ○ | 0 | なし |
| 生ウサギ肉 | +40kcal | ◎ | 0 | ○ | 食中毒30% |
| 焼きウサギ肉 | +55kcal | ◎ | 0 | ○ | なし |
| 缶詰（野菜） | +30kcal | △ | ○ | 0 | なし |
| 缶詰（魚） | +50kcal | ◎ | 0 | ○ | なし |
| 釣った魚（焼き） | +60kcal | ◎ | 0 | ◎ | なし |
| ヤシの実 | +35kcal | 0 | ○ | ◎ | なし |

**体温**
| 要因 | 変化（℃/分） |
|-----|------------|
| 夜間（気温15℃） | -0.3 |
| 雨中（気温15℃） | -0.6 |
| 嵐（気温10℃） | -1.2 |
| 焚き火の近く | +0.5 |
| シェルター内 | -0.1（断熱） |
| 走る | +0.2 |
| 夏昼間 | +0.1 |
| 防寒服装着 | -0.05（断熱） |

**睡眠**
- 毎10分 -3
- 眠る（シェルター内）で +50/時間
- 野外で眠ると +20/時間（効率低下、悪夢リスク）
- 50以下：採集効率-20%
- 20以下：視野ぼやけ、操作精度-30%
- 0：幻覚・行動不能

**精神衛生**
- 毎日 -5（孤独）
- 焚き火の傍 +3/分
- ヒント発見 +20
- 仲間の死（将来の協力モード）-50
- 狼に追われた直後 -15
- 廃村の怖い日記を読む -10
- 30以下：幻聴
- 10以下：パニック（ランダム入力が発生）

## 7. 状態異常システム

| 状態異常 | 発生条件 | 症状 | 治療 |
|---------|---------|------|------|
| **出血** | 動物の攻撃・鋭利な物への接触 | HP -2/秒 | 布で包帯 |
| **深傷** | 出血を30秒放置 | HP -4/秒 + 移動速度-20% | 包帯＋消毒 |
| **感染症** | 深傷の放置・錆びた金属接触 | 発熱・HP -1/秒・体温+1 | 薬草煎じ薬 or 抗生物質 |
| **敗血症** | 感染症を24ゲーム時間放置 | HP -5/秒・48時間で死亡 | 抗生物質のみ（廃村で入手） |
| **骨折** | 高所落下・重い攻撃 | 移動速度-50%・採集効率-50% | 添え木クラフト（木×2）＋安静 |
| **食中毒** | 生肉・汚染水 | 嘔吐（水分-30）・カロリー吸収-80% | 活性炭（廃村）or 安静 |
| **低体温症Lv1** | 体温35℃以下 | 移動速度-20%・手が震える演出 | 焚き火・シェルター |
| **低体温症Lv2** | 体温33℃以下 | 操作不能になる瞬間が発生 | 緊急焚き火が必要 |
| **低体温症Lv3** | 体温31℃以下 | 意識朦朧・数分で死亡 | 即座に暖を取らないと不可逆 |
| **熱中症** | 体温39℃以上 | 水分消費2倍・頭痛演出 | 日陰・水分補給 |
| **壊血病** | ビタミンC不足（14日間） | HP上限-20・歯茎出血エフェクト | 果物・野菜の摂取 |
| **筋痙攣** | 塩分不足（10日間） | ランダムで一時的移動不能 | 海藻・塩分補給 |
| **筋力低下** | タンパク不足（7日間） | 採集量-30%・速度-10% | 肉・魚の摂取 |
| **脱水症** | 水分20以下 | 視野ぼやけ・判断力低下 | 飲水 |
| **疲労** | 睡眠50以下 | 採集効率-20%・速度-10% | 睡眠 |
| **悪夢** | 野外睡眠・精神衛生30以下 | 睡眠効率-50%・精神衛生-10 | 屋内で眠る |

## 8. 傷の進行システム

```
【受傷】動物攻撃 / 鋭利な物
  ↓ 放置30秒
【出血】HP -2/秒
  ↓ 放置30秒
【深傷】HP -4/秒、移動-20%
  ↓ 放置 (ゲーム内6時間)
【感染症】発熱・HP -1/秒
  ↓ 放置 (ゲーム内24時間)
【敗血症】HP -5/秒 → 48時間で確実に死亡
```

**治療アイテム：**
- 布（衣類から作成）→ 出血・深傷の包帯
- 消毒液（廃村の薬局）→ 感染リスク軽減
- 薬草煎じ薬（島の薬草）→ 感染症初期に対応
- 抗生物質（廃村の薬局）→ 感染症・敗血症に対応

## 10. 天候システム

### 天候の遷移
```
晴れ ←→ 曇り ←→ 雨 ←→ 嵐
  （各天候はランダム持続時間、季節で確率変化）
```

| 天候 | 体温変化 | 焚き火 | 視界 | 狼の活性 |
|-----|--------|-------|-----|--------|
| 晴れ | ±0 | 通常 | 100% | 夜のみ |
| 曇り | -0.1 | 通常 | 80% | 夜のみ |
| 雨 | -0.6 | 薪消費2倍 | 60% | 夜＋雨中も活動 |
| 嵐 | -1.2 | 維持困難（薪3倍消費、失敗で消える） | 30% | 嵐中は隠れる |

### 嵐の予兆（リアリズム演出）
- 動物が突然逃げ始める（嵐の10分前）
- 気圧低下エフェクト（画面周縁が暗くなる）
- 鳥の鳴き声が消える

## 11. 火のリアリズム

| 薪の状態 | 着火難易度 | 燃焼時間 |
|---------|---------|--------|
| 乾いた木材 | 簡単 | 通常 |
| 湿った木材 | 困難（失敗率50%） | 短い |
| 雨ざらしの木材 | 非常に困難（失敗率80%） | 非常に短い |

- 木材の乾燥：シェルター内に保管で24ゲーム時間後に乾燥
- 嵐中：屋根なしの焚き火は消える可能性がある
- 風向き：焚き火の煙は風下に流れ、動物が察知することがある

## 12. 生態系・食物連鎖

```
ベリー・草 → ウサギ・シカ → 狼（捕食）
              ↓ 乱獲
         個体数減少
              ↓
         狼の食料不足
              ↓
         プレイヤーを昼間も積極的に狙う
```

| 動物の個体数 | 狼の行動変化 |
|-----------|-----------|
| ウサギ 20匹以上 | 夜のみ活動 |
| ウサギ 10〜19匹 | 夜＋雨中に活動 |
| ウサギ 5〜9匹 | 昼夜問わず活動 |
| ウサギ 0〜4匹 | 群れで行動・攻撃力1.5倍 |

### 動物の感覚システム
- **嗅覚**：風上からアプローチすると気づかれない（ステルス狩猟可能）
- **聴覚**：走ると半径200px以内の動物が逃げる（歩けば気づかれにくい）
- **視覚**：夜間の視野半径が昼間の60%（月明かりで変動）

| 動物 | 行動 | 危険度 | 入手素材 |
|-----|------|--------|--------|
| **ウサギ** | 逃げる。罠で捕獲可 | なし | 肉・毛皮 |

## 14. 潮汐システム

- **サイクル**：ゲーム内12時間ごとに満潮↔干潮が切り替わる
- **干潮時**：南東の珊瑚礁エリアに渡れる → ヘリ用燃料タンクが入手可能
- **満潮時**：南西の干潟の一部が水没 → いかだ発進ポイントが変わる
- **嵐＋満潮**：通常陸地の一部まで浸水（プレイヤーが体に水をかぶると体温低下）

## 15. 廃村「オリハル集落」

### エリア構成
| 建物 | 内容 | 危険 |
|-----|------|------|
| 民家A〜F | 缶詰・衣類・日記・工具 | 崩落リスク（重い物取ると瓦礫） |
| 診療所 | 消毒液・抗生物質・包帯・添え木 | なし |
| 雑貨屋 | 缶切り・ライター・ロープ | なし |
| 漁師小屋（港） | 釣り竿・朽ちた船・航海図 | 床が腐敗（落下リスク） |
| 発電所 | 発電機・電子部品・ガソリン缶 | 感電リスク（修理中） |
| 無線塔 | アンテナパーツ・無線マニュアル | 高所作業リスク |

### 廃村の謎（手がかりアイテム）
- 日記A：「島の北に何かが落ちてきた…」
- 電報：「全員避難せよ。理由は言えない」
- 手紙：「この島には帰らない方がいい」
- 写真：笑顔の家族。裏に「1987年 オリハル集落」
- 最後の日記：（廃村の真相、隠しエリアで入手）

### 廃村の電力系統
```
発電機修理（燃料＋エンジン部品）
  ↓
廃村の電力復旧
  ↓
水道ポンプ稼働 → 安定した水源確保（水分補給が劇的に楽に）
無線機稼働 → SOSエンディング解放
倉庫の電動シャッター開放 → 特殊素材入手
```

## 17. 5つのエンディング

| # | エンディング | 難易度 | 主な条件 | ゲーム内時間目安 |
|---|------------|-------|---------|--------------|
| 1 | **いかだで脱出** | ★★ | 丸太×8 + 蔓×6 + 葉帆×2 を建造 | 5〜10日 |
| 2 | **SOS無線** | ★★★ | 発電機修理 + 電子部品 + アンテナ修理完了 | 15〜25日 |
| 3 | **船を修理して出航** | ★★★★ | 港の船修理 + エンジン部品 + 燃料 + 航海図 | 20〜35日 |
| 4 | **ヘリを修理して飛ぶ** | ★★★★★ | 山頂の墜落ヘリ修理 + ローター + 燃料（珊瑚礁で入手） + 操縦マニュアル | 30〜50日 |
| 5 | **定住を選ぶ** | ★★★★★ | 農場×3 + 安定水源 + 恒久シェルター + 30日生存 | 30日 |
