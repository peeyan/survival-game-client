## 4. 子供の追加特性システム（コア設計）

### 設計思想

> **「誰と行動したかが、子供を変える。」**

ソウとハナは「成長型キャラクター」。
父と過ごすほどエンジニア的素養が、母と過ごすほど医療・自然的素養が育つ。
この追加特性が、**どのENDに到達できるかを直接左右する**。

---

### 追加特性ゲージの仕組み

```
【ソウの適性ゲージ（例）】

父寄り ████████░░ 80%
母寄り ██░░░░░░░░ 20%

→ 現在の追加特性：エンジニアLv3 + メディカルLv1
```

| ゲージ変動のトリガー | 変化量 |
|------------------|-------|
| 父と2人で採掘・建築・修理をする | 父寄り +5 |
| 母と2人で採集・料理・治療をする | 母寄り +5 |
| 単独で廃墟探索をする | 父寄り +3（構造への興味） |
| 単独で動物観察をする | 母寄り +3（自然への親しみ） |
| 父の指示で動く | 父寄り +2 |
| 母に助けてもらう | 母寄り +2 |

**判定タイミング：** ゲーム内3日ごとに現在のゲージを確認し、追加特性のレベルを更新する。

## 7. 複数人作業システム

### 設計思想

> **「一人では辿り着けないENDがある。家族全員で協力したとき、最高の選択肢が開く。」**

### 作業別・必要人数と条件

| 作業 | 必要人数 | 必須キャラ | 条件 |
|-----|---------|---------|------|
| 大型いかだ建造 | 2人以上 | ケン必須 | ― |
| 翼竜の巣台建造 | 2人以上 | ハナ必須 | ハナ 父寄りLv2 or 母寄りLv3 |
| 発電機修理 | 2人 | ケン必須 | ソウ or ハナが父寄りLv1以上 |
| 車修理 | 3人以上 | ケン必須 | ソウ父寄りLv3以上 |
| ドローン修理 | **全員（4人）** | **全員必須** | ケン + ソウ父Lv4 + ハナ父Lv3 |
| 草食恐竜に乗る | 2人以上 | ハナ必須 | ハナ 母寄りLv3 |
| 翼竜飛行 | 全員 | ハナ必須 | ハナ 母寄りLv3 |
| 永住END宣言 | 全員 | 全員必須 | ― |
| 王国END宣言 | 全員 | 全員必須 | ― |

### 「全員作業」の重み

**ドローン修理は4人全員が揃って初めて完成する。**

```
ケン：　エンジン・電気系統の修理
ソウ：　重い部品の運搬・補助修理（Lv4）
ハナ：　精密な配線作業（小さな手が入る場所）（Lv3）
アキ：　作業中の体調管理・食料補給・チームのまとめ
```

**誰か一人でも欠けると、ドローンは完成しない。**
これが「家族全員で生き延びること」の最大のインセンティブになる。

## 8. ENDとキャラクター特性の対応表

| END | 必須特性 | 到達不可になる条件 |
|-----|---------|----------------|
| 🛶 川いかだ | ケンのみ | ケン死亡 |
| 🦛 草食恐竜 | ハナ母Lv3 | ハナが父寄りに育った / ハナ死亡 |
| 🏔 山越え | ― | ― |
| 🦕 翼竜飛行 | ハナ母Lv3 | 上記と同様 |
| 🚗 車修理 | ケン + ソウ父Lv3 | ケン or ソウ死亡 / ソウが母寄りに育った |
| 🚁 ドローン | **全員** + ソウ父Lv4 + ハナ父Lv3 | **誰か1人でも死亡で不可** |
| 🏡 永住END | 全員生存 | 誰か死亡 |

| ✨ 謎のEND | 未定（探索系） | ― |

### 「誰も死なせない」動機

上の表が示すとおり、**全員生存**はゲームで最も多くのENDを解放する条件。
特にドローンENDは「全員生存 + 子供たちが正しく育った」という二重の条件。

> 家族を守ることが、最高の選択肢への唯一の道。
