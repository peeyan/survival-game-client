## 1. ゲームの本質的な問い

> **「この世界で生き延びた先に、何を選ぶか。」**
> **「そして——なぜ、ここに都市の廃墟があるのか。」**

帰る方法はわからない。謎は謎のまま。
白亜紀の密林の奥に、荒廃した現代都市がある。
誰が作ったのか。なぜここにあるのか。答えは、ない。
でも家族には選択肢がある。

- **前へ進む** ── 古代の知恵で、あるいは廃墟から拾った技術で
- **ここに根を張る** ── この世界を受け入れ、王国を築く

## 2. エンディングシステム

### 設計原則

> **エンドは「条件達成で自動発動」しない。条件が揃ったとき、プレイヤーに選択肢が提示されるだけ。**

いかだを作っても出発しなくていい。永住条件を満たしても定住しなくていい。
王国ENDの要件を達成してもまだ旅立ちを選べる。
**複数の条件を同時に満たしてから、最後に好きなENDを選ぶ** ことも可能。

---

### エンディング一覧（World 01）

| エンド | 名前 | トリガー | スコアボーナス | 感情トーン |
|-------|------|---------|------------|---------|
| 🛶 | **旅立ちEND** | いかだ乗船中に「出発する？」YES | +5,000pt | 期待・未知・つづく |
| 🏡 | **永住END** | 永住条件達成後に「ここに永住する？」YES | +3,000pt | 平和・愛着・切なさ |
| 🏰 | **王国END** | 王国条件達成後に「この地に王国を築く？」YES | +8,000pt | 誇り・壮大・歴史の始まり |
| ✨ | **謎のEND（隠し）** | 壁画5枚収集＋特定行動 | 全スコア×3.0 | 驚き・謎・答えなし |

**難易度序列：** 旅立ち（易） ＜ 永住（中） ＜ 王国（難） ＜ 謎（超難）

## 3. 旅立ちEND 詳細（World 01）

### コンセプト

> 「帰り方はわからない。でも、ここにいても何も変わらない。前へ進もう。」

### 設計原則

- どの手段でも「旅立ちEND」としてカウント。次のワールド（World 02）へ繋がる
- 到着地点が手段ごとに異なり、World 02 の序盤体験が変わる
- **作っても・なついても強制出発しない**。使ったとき・乗ったときに「出発する？」を確認する

---

### 旅立ち手段 一覧

| 手段 | 難易度 | World 02 到着地 | 感情トーン |
|-----|--------|----------------|---------|
| 🛶 川いかだ | ★★☆☆☆ | 海岸（開けた浜辺） | 穏やか・希望 |
| 🦕 翼竜飛行 | ★★★★☆ | 山岳高地（急斜面） | 壮大・感動 |
| 🏔 山越え | ★★★☆☆ | 広大な平原 | 達成感・疲弊 |
| 🦛 草食恐竜に乗る | ★★★☆☆ | 草原（平坦） | ロマン・温かさ |

## 4. 永住END 詳細（World 01）

### コンセプト

> 「もう探すのをやめた、ということじゃない。ここが、好きになったんだ。」

### 達成条件

```
① 石造りの家を建設する（木の家より上位）
② 農地を3区画以上完成させる（種を植えて収穫まで）
③ 防衛柵でキャンプ周囲を囲う（柵×30以上の連続設置）
④ Day 20 以上生存する

→ 条件達成で「永住する？」の選択肢が出現する
   → YES → 永住END
   → NO  → そのまま続けてサバイバル・いかだを作って旅立つ選択も残る
```

**重要：条件を満たしても強制エンドにはならない。**
「永住できる状態になった」だけで、まだ旅立ちを選べる。

### 永住ENDシーケンス

```
夕暮れ。完成した村。
煙突から煙が出ている。
農地では作物が揺れている。

家族が丘の上から村を見渡す。

アキ：「……きれいね」
ハナ：「ねえ、恐竜たちに名前つけていい？ご近所さんだもん」
ソウ：「（照れながら）……まあ、悪くないかな」
ケン：「（長い間があって）……ありがとう、みんな」

星が出る。
焚き火の光が揺れる。
恐竜の遠吠えが、もう怖くない。

─────────────────────────────
    ここが、私たちの家になった。
─────────────────────────────
```

→ リザルト画面 → 村の「その後」の静止画が表示される

**設計意図：** 永住ENDは「諦め」ではなく「選択」。強さの到達点として最も美しいエンド。

## 5. 王国END 詳細（World 01）

### コンセプト

> 「帰ることも、どこかへ行くことも選ばなかった。この時代に、文明を作ることを選んだ。」

最も達成困難なエンド。大量の建材・農業・防衛を完成させた者だけが見られる。

### 達成条件

```
① 石造りの家の壁を 40 枚以上、連続して設置する
   （= しっかりした「集落」の規模）
② 農地を 5 区画以上完成させ、すべてで収穫を終える
③ 石の柵（Tier2）で集落の周囲を 50 枚以上囲う
④ 燻製小屋・調理台をそれぞれ設置する
⑤ Day 30 以上生存する

→ 条件達成で「この地に王国を築く？」の選択肢が出現する
   → YES → 王国END
   → NO  → 引き続きサバイバル・旅立ちも選べる
```

### 王国ENDシーケンス

```
俯瞰ショット。集落全体が映る。
石造りの家々。農地に実る作物。松明の明かり。
柵の外では恐竜たちが遠巻きに歩いている。

夜明け。ケンが丘の上に立つ。

ケン：「……俺たちは、帰れなかった」
アキ：「（隣に立つ）うん」
ソウ：「でも、これ作ったじゃん。俺たちが」
ハナ：「ここ、すきだよ。ずっといたい」

長い沈黙。

ケン：「……そうだな」

─────────────────────────────
  紀元前7000万年。この地に、
  人類最初の王国が誕生した。
─────────────────────────────
```

## §3A. 荒廃都市エリア（World 01 隠しエリア）

### コンセプト

> 「白亜紀の密林の奥に、荒廃した現代都市がある。恐竜がビルの間を歩いている。誰がここに都市を作ったのか——答えは、ない。」

最大の謎。最高の報酬。最悪の危険。

---

### エリア概要

| 項目 | 内容 |
|-----|------|
| 位置 | マップ北東（深い森の奥・最危険エリアの先） |
| 到達難易度 | ★★★★★（ラプター密集地帯を突破した先） |
| 雰囲気 | 蔦に覆われたビル群・錆びた車・割れたガラス・コンクリート瓦礫 |
| 敵 | ラプター（通常より大きく強い個体）＋崩落する床・不安定な足場 |
| BGM | 廃墟専用：静寂と風音。ときおり金属が軋む音 |

### ビジュアルイメージ

```
ジャングルから突然、コンクリートの壁が現れる。
窓ガラスは割れ、蔦がビルを包んでいる。
道路の割れ目から草木が生えている。
錆びた車が転がっている。信号機が傾いている。
その間を、ラプターが歩いている。

ソウ：「……なんだ、これ」
ケン：「（声が出ない）」
ハナ：「まちだ。まちがある」
```

---

### 廃墟内の探索・収集物

| 収集物 | 場所 | 用途 |
|-------|------|------|
| 廃車パーツ | 道路上の廃車 | 車修理の素材 |
| 電子部品 | ビル内の電子機器の残骸 | ドローン修理の素材 |
| ガソリン缶（空） | ガソリンスタンド跡 | 精製ガソリンの容器 |
| メカニクス教本 | ビル内の書棚 | 車/バイクの修理方法を習得 |
| 電気回路図 | ビル内の書棚 | ドローン/発電機の修理方法を習得 |
| 現代人のメモ | 各所に散在 | 物語の断片（「先人がいた」証拠） |
| 壊れた時計 | 複数箇所 | すべて同じ時刻で止まっている（謎） |
| 家族写真（他人） | アパート跡 | 「別の家族もここにいた」ことを示す |

---

### 現代技術ツリー

廃墟エリアで「教本」を読むことで技術を習得できる。習得した技術は現世界でのサバイバルにも使える。

```
【技術ツリー】

発電機修理（電気回路図 × 1 + 廃車パーツ × 5 + 電子部品 × 3）
    │
    ├──→ 電力供給（廃墟の一部に電気が通る）
    │         └──→ 電動調理器・電動採掘機（超効率）
    │
    └──→ ドローン修理が解放される
              └──→ 🚁 ドローン旅立てルート

メカニクス教本を読む
    │
    ├──→ バイク修理（廃車パーツ × 10）
    │         └──→ フィールド高速移動
    │
    └──→ 車修理が解放される（廃車パーツ × 20 + ガソリン精製）
              └──→ 🚗 車旅立てルート
```

---

### 車の修理工程（「とてつもなく大変」の実際）

```
STEP 1: 廃墟エリアに到達する
         → ラプター大群を突破。最低でも強力な武器・松明×10が必要

STEP 2: メカニクス教本を発見・読破する
         → 廃墟ビル3階。崩落リスクのある不安定な床を越えた先

STEP 3: 廃車パーツを20個収集する
         → 廃車1台から2〜3個しか取れない。廃墟内10台以上を解体

STEP 4: ガソリンを精製する
         → 空のガソリン缶 + 特定の植物油 × 10 + 焚き火での加工
         → 植物油は草食草原の特定植物から採取（少量しか取れない）

STEP 5: エンジンを修理する
         → 廃車パーツ × 20 + ガソリン × 5 + 電子部品 × 2
         → 修理に「ゲーム内24時間（リアル約2.4分）」かかる

STEP 6: 試運転
         → 廃墟エリア内の広場で動作確認
         → この時点で「旅立つ？」が選択可能に
```

**総所要時間の目安：** 廃墟エリア到達まで Day 15〜20 / 車完成まで Day 25〜35

---

### ドローンの修理工程（最高難易度）

```
STEP 1〜3: 車修理と同じ（廃墟到達・教本読破）

STEP 4: 電気回路図を発見する
         → 廃墟ビル最上階（最も危険な場所）

STEP 5: 発電機を修理・稼働させる
         → 廃車パーツ × 5 + 電子部品 × 3 → 廃墟の一部に電力供給

STEP 6: ドローンの電子部品を収集する
         → ビル各フロアに散在。最も希少な部品は地下室に
         → 地下は暗くてラプターが潜んでいる（松明必須）

STEP 7: ドローン本体を修理する
         → 電子部品 × 15 + 廃車パーツ × 5 + 発電機稼働状態が必要

STEP 8: 屋上へ運ぶ
         → ドローンは重い。複数回に分けて運搬（インベントリ制限あり）

STEP 9: 起動テスト → 「旅立つ？」
```

**総所要時間の目安：** Day 30〜50。ゲームで最も長い達成ルート。

---

### 廃墟エリアが語る謎（環境ストーリーテリング）

| 発見物 | プレイヤーが感じること |
|-------|-------------------|
| 壊れた時計がすべて同じ時刻（11:47）で止まっている | 「何かが起きた瞬間がある」 |
| 家族写真（父・母・子供2人の4人家族） | 「……私たちと同じ構成だ」 |
| 「また来てしまった」と書かれたメモ | 「この人は、ここに複数回来たのか？」 |
| 誰かが書いた日記の最後のページ（「今日で○○日目。帰り方はわからない」） | 「先人は帰れたのか」 |
| 廃墟の壁に彫られた「→」の矢印 | 「誰かが道を示そうとした」 |

**これらは絶対に説明しない。** プレイヤーの想像の余地として残す。

## 7. ゲームオーバー（死亡演出）

### 設計方針

死因によって演出・テキスト・BGMが変わる。
「どうやって死んだか」がプレイヤーの記憶に残り、次のプレイの緊張感を高める。

### 死因別演出一覧（World 01）

| 死因 | 発動条件 | 演出 | テキスト |
|-----|---------|------|---------|
| **空腹死** | 空腹度0→HP0 | 画面がゆっくり暗転。フェードアウト | 「空腹で、力が尽きた…」 |
| **ラプター死** | ラプターにHP0にされる | 赤フラッシュ→急速暗転。恐竜の咆哮SE | 「暗闇の中で、足音が止まった…」 |
| **（将来）毒死** | 毒状態でHP0（World 02） | 画面が緑がかって歪む | 「視界が、ぼやけていく…」 |
| **（将来）凍死** | 体温0→HP0（World 03） | 画面が白くなっていく | 「寒さが、痛みに変わった…」 |

### ゲームオーバー画面の構成

```
┌────────────────────────────────────┐

│   （死因に応じたアイコン）           │
│                                    │

│   ─────────────────────────────   │
│   生存日数：  X日  XX時間            │
│   スコア：    XXX pt                │
│   ─────────────────────────────   │
```

## 13. 余白の美学（語りすぎないために）

### 絶対に説明しないこと

- なぜ川を下ると次のワールドへ行けるのか
- 家族がいつ、本当の意味で「帰れる」のか
- 永住ENDのその後（家族は一生この世界に？子孫は？）
- 謎のENDの先に何があるのか

### プレイヤーに委ねる問い

> 「旅立った先に、何があるんだろう。」
> 「永住した家族は、幸せだったのかな。」
> 「あの川は、どこまで続いているんだろう。」
