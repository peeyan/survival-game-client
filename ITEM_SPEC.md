## アイテム共通パラメータ定義

| パラメータ | 説明 |
|-----------|------|
| **ティア** | 1=序盤/2=中盤/3=終盤・廃墟 |
| **スタック** | インベントリ1枠に入る最大個数 |
| **腐敗** | ゲーム内日数。経過後は使用不可（食料のみ） |
| **耐久** | 使用回数。0になると消滅（道具・武器のみ） |
| **重量** | インベントリへの負荷。重いものは枠を複数使用 |
