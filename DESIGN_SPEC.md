## 1. ビジュアルコンセプト

### キャラクター方向性

**Stardew Valley 風のかわいいデフォルメ。**

| 要素 | 方針 |
|-----|------|
| キャラサイズ | 32×32px（現行）を継続 |
| 頭身 | 2頭身〜2.5頭身。頭が大きく、目が丸い |
| 恐竜 | 実物より丸くかわいい。ただし夜のラプターは眼が光る（怖さを演出） |
| カラーパレット | 白亜紀 = 深緑・黄緑・茶・橙。夜 = 紺・黒・黄（焚き火） |
| フォント | ドット系 or 丸ゴシック系。英語は Pixel font |

|------------|------|------------|

### 4-7. スコア計算

```
総スコア = (生存日数 × 100)
         + (ヒント収集数 × 500)
         × 脱出倍率

脱出倍率：
  脱出しなかった場合：× 1.0
  脱出成功：         × 2.0
  全員生存脱出：     × 3.0（協力時）
```

| ケース | スコア例 |
|-------|---------|
| 3日生存・脱出なし | 300pt |
| 7日生存・ヒント3個・脱出なし | 700 + 1,500 = 2,200pt |
| 7日生存・ヒント5個・脱出成功 | (700 + 2,500) × 2 = 6,400pt |

### 4-8. 難易度カーブ

| 日数 | 空腹管理 | ラプター | 難易度感 |
|-----|---------|---------|---------|
| Day 1 | ベリーで十分 | 1匹・遅い | チュートリアル |
| Day 2〜3 | 焚き火が必要 | 2匹 | 慣れてきた |
| Day 4〜7 | 安定した食料確保が鍵 | 3匹・速い | 本番 |
| Day 8+ | 高効率クラフト必須 | 5匹 | 上級者向け |
